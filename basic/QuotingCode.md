# Quoting Code
## Single backticks
- Text
    ```
    Use `git status` to list issue
    ```
- Go
    > Use `git status` to list issue
## Triple backticks
- Text
    ```
        ```
        git status
        git add
        git commit
        ```
    ```
- Go
    >    ``` bash
    >    git status
    >    git add
    >    git commit
    >    ```
